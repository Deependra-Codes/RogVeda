# Records

This directory stores the durable artifacts that back non-trivial changes.

## Layout

- `intents/`
  why a change exists
- `research/`
  options, tradeoffs, sources, and recommendation
- `specs/`
  concrete behavior and acceptance shape
- `validations/`
  what was actually checked and what remains uncertain
